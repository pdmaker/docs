## 使用建议

### 开发环境

- 使用便宜的模型进行测试（如 GPT-3.5-turbo）
- 验证逻辑正确后切换到高质量模型

### 生产环境

- 根据实际需求选择合适的模型
- 监控使用量和成本
- 设置合理的并发限制

### 优化建议

1. **合理设置参数**
   - temperature: 控制创造性
   - max_tokens: 限制输出长度
   - top_p: 控制多样性

2. **使用流式输出**
   - 提升用户体验
   - 允许提前终止

3. **缓存常见请求**
   - 减少 API 调用
   - 降低成本

4. **错误处理**
   - 实现重试机制
   - 准备降级方案
