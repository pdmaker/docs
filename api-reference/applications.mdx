## 代码编辑器集成

### 🎯 Cursor - AI代码编辑器

Cursor是最受欢迎的AI代码编辑器之一，完美支持API易。

#### 🔧 配置步骤

<Steps>
  <Step title="打开Cursor设置">
    在Cursor中按 `Cmd/Ctrl + ,` 打开设置页面
  </Step>
  <Step title="找到Models配置">
    在设置中找到 "Models" 或 "AI" 相关选项
  </Step>
  <Step title="添加自定义API">
    点击 "Add Model" 或 "Custom" 选项
  </Step>
  <Step title="配置API易服务">
    填入以下配置信息：
    - **API Base URL**: `https://vip.apiyi.com/v1`
    - **API Key**: 你的API易密钥
    - **Model**: 选择你想使用的模型
  </Step>
</Steps>

## 开发工具集成

### 🐙 GitHub Copilot

虽然GitHub Copilot不能直接修改API端点，但你可以使用API易构建类似的功能。

#### 自定义Copilot

```python
# 创建自定义代码补全工具
import openai

class CustomCopilot:
    def __init__(self, api_key):
        self.client = openai.OpenAI(
            api_key=api_key,
            base_url="https://vip.apiyi.com/v1"
        )
    
    def complete_code(self, code_context, cursor_position):
        """代码补全"""
        prompt = f"""
        你是一个专业的编程助手，请为以下代码提供合适的补全：
        
        代码上下文：
        {code_context}
        
        光标位置：{cursor_position}
        
        请提供最可能的代码补全，只返回补全的代码，不要包含解释。
        """
        
        response = self.client.chat.completions.create(
            model="gpt-4o",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.2,
            max_tokens=200
        )
        
        return response.choices[0].message.content
    
    def explain_code(self, code):
        """解释代码"""
        prompt = f"""
        请解释以下代码的功能和原理：
        
        {code}
        
        请提供简洁明了的解释。
        """
        
        response = self.client.chat.completions.create(
            model="gpt-4o",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.3,
            max_tokens=500
        )
        
        return response.choices[0].message.content

# 使用示例
copilot = CustomCopilot("YOUR_API_KEY")
completion = copilot.complete_code("def fibonacci(n):", 18)
explanation = copilot.explain_code("lambda x: x**2")
```

### 📊 监控和调试

#### 请求日志记录

```javascript
class APILogger {
  constructor() {
    this.logs = [];
  }
  
  log(request, response, duration) {
    const logEntry = {
      timestamp: new Date().toISOString(),
      model: request.model,
      inputTokens: this.estimateTokens(request.messages),
      outputTokens: this.estimateTokens(response.choices[0].message.content),
      duration: duration,
      cost: this.calculateCost(request.model, inputTokens, outputTokens)
    };
    
    this.logs.push(logEntry);
    console.log(`API调用: ${request.model}, 耗时: ${duration}ms, 成本: $${logEntry.cost}`);
  }
  
  calculateCost(model, inputTokens, outputTokens) {
    const pricing = {
      'gpt-4o': { input: 0.005, output: 0.015 },
      'gpt-4o-mini': { input: 0.00015, output: 0.0006 },
      'claude-3-5-sonnet-20241022': { input: 0.003, output: 0.015 }
    };
    
    const modelPricing = pricing[model] || pricing['gpt-4o-mini'];
    return (inputTokens * modelPricing.input + outputTokens * modelPricing.output) / 1000;
  }
}
```
