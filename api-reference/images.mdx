## 接口信息

**接口地址**：`POST https://api.apiyi.com/v1/images/generations`

**兼容性**：完全兼容OpenAI官方API格式

## 请求参数

### 必需参数

| 参数 | 类型 | 说明 |
|------|------|------|
| `model` | string | 图像生成模型，如 `dall-e-3`、`gpt-4o-image` |
| `prompt` | string | 图像描述提示词，建议使用英文 |

### 可选参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `n` | integer | 1 | 生成图片数量 (1-10) |
| `size` | string | "1024x1024" | 图片尺寸 |
| `quality` | string | "standard" | 图片质量：standard/hd |
| `style` | string | "vivid" | 图片风格：vivid/natural |
| `response_format` | string | "url" | 响应格式：url/b64_json |
| `user` | string | null | 用户标识符 |

### 支持的尺寸

不同模型支持的尺寸规格：

| 模型 | 支持尺寸 |
|------|----------|
| DALL-E 3 | 1024x1024, 1024x1792, 1792x1024 |
| DALL-E 2 | 256x256, 512x512, 1024x1024 |
| GPT-4o Image | 1024x1024, 1024x1792, 1792x1024 |
| Flux 系列 | 1024x1024, 768x1344, 1344x768 |
| Midjourney | 1024x1024, 1152x896, 896x1152 |

<CodeGroup>


```javascript Node.js
import OpenAI from 'openai';

const openai = new OpenAI({
  apiKey: 'YOUR_API_KEY',
  baseURL: 'https://api.apiyi.com/v1'
});

async function generateImage() {
  const image = await openai.images.generate({
    model: 'gpt-4o-image',
    prompt: 'A beautiful sunset over mountains with a lake in the foreground',
    n: 1,
    size: '1024x1024',
    quality: 'hd'
  });

  console.log(image.data[0].url);
}

generateImage();
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://api.apiyi.com/v1"
)

response = client.images.generate(
    model="gpt-4o-image",
    prompt="A beautiful sunset over mountains with a lake in the foreground",
    n=1,
    size="1024x1024",
    quality="hd"
)

print(response.data[0].url)
```

```go Go
package main

import (
    "context"
    "fmt"
    "github.com/sashabaranov/go-openai"
)

func main() {
    config := openai.DefaultConfig("YOUR_API_KEY")
    config.BaseURL = "https://api.apiyi.com/v1"
    
    client := openai.NewClientWithConfig(config)
    
    resp, err := client.CreateImage(
        context.Background(),
        openai.ImageRequest{
            Model:   "gpt-4o-image",
            Prompt:  "A beautiful sunset over mountains with a lake in the foreground",
            N:       1,
            Size:    openai.CreateImageSize1024x1024,
            Quality: openai.CreateImageQualityHD,
        },
    )
    
    if err != nil {
        fmt.Printf("Error: %v\n", err)
        return
    }
    
    fmt.Println(resp.Data[0].URL)
}
```

### 2. 并发请求

API易支持高并发图片生成：

```python
import asyncio
import aiohttp

async def generate_image(session, prompt):
    async with session.post(
        'https://api.apiyi.com/v1/images/generations',
        headers={'Authorization': 'Bearer YOUR_API_KEY'},
        json={
            'model': 'gpt-4o-image',
            'prompt': prompt,
            'size': '1024x1024'
        }
    ) as response:
        return await response.json()

async def batch_generate():
    prompts = [
        "A cat sitting on a windowsill",
        "A dog playing in a park",
        "A bird flying in the sky"
    ]
    
    async with aiohttp.ClientSession() as session:
        tasks = [generate_image(session, prompt) for prompt in prompts]
        results = await asyncio.gather(*tasks)
    
    return results
```

## 错误处理

### 常见错误

| 错误码 | 说明 | 解决方案 |
|--------|------|----------|
| 400 | 提示词违规或参数错误 | 检查提示词内容和参数格式 |
| 401 | API Key无效 | 验证API Key是否正确 |
| 429 | 请求过于频繁 | 降低请求频率，使用队列处理 |
| 500 | 服务器错误 | 稍后重试或联系支持 |

### 内容政策

为确保服务稳定，请避免生成以下内容：
- 暴力、血腥内容
- 政治敏感内容
- 色情、成人内容
- 侵犯版权的内容
- 涉及真实人物的不当内容
