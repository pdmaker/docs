# Bob 翻译集成指南

Bob 是 macOS 上一款优秀的翻译软件，支持划词翻译、截图翻译等功能。通过集成 API易，您可以使用 AI 模型提供更准确、更自然的翻译结果。

## 安装 Bob

如果您还未安装 Bob，请先下载安装：

1. 访问 [Bob 官网](https://bobtranslate.com)
2. 下载最新版本
3. 安装并打开 Bob

## 配置 API易

### 步骤 1：获取 API 密钥

1. 登录 [API易控制台](https://vip.apiyi.com)
2. 在控制台页面复制您的 API 密钥

### 步骤 2：添加 OpenAI 翻译服务

1. 打开 Bob 设置（点击菜单栏图标 > 偏好设置）
2. 切换到"服务"标签页
3. 如果没有 OpenAI 翻译服务，点击"+"添加
4. 选择"OpenAI 翻译"

### 步骤 3：配置服务参数

在 OpenAI 翻译服务配置中：

- **API Key**：粘贴您的 API易 密钥
- **API URL**：`https://api.apiyi.com`
- **模型**：选择或输入要使用的模型（如 `gpt-3.5-turbo`）

### 步骤 4：测试配置

1. 点击"测试"按钮验证配置
2. 如果显示成功，说明配置正确
3. 点击"保存"

## 使用方法

### 划词翻译

1. 选中需要翻译的文本
2. 按下快捷键（默认 `⌥ + D`）
3. Bob 会弹出翻译结果

### 截图翻译

1. 按下截图快捷键（默认 `⌥ + S`）
2. 框选需要翻译的区域
3. Bob 会识别并翻译图片中的文字

### 输入翻译

1. 调出 Bob 窗口（默认 `⌥ + Space`）
2. 输入或粘贴要翻译的文本
3. 选择目标语言
4. 查看翻译结果

## 高级配置

### 自定义提示词

优化翻译质量的提示词模板：

```text
你是一位专业的翻译专家，精通多国语言。请将以下{source_lang}文本翻译成{target_lang}。

要求：
1. 保持原文的语气和风格
2. 使用地道的表达方式
3. 对于专业术语，在翻译后用括号标注原文
4. 注意文化差异，适当调整表达

原文：{text}
```

### 模型选择建议

不同场景推荐使用的模型：

| 使用场景 | 推荐模型 | 特点 |
|---------|---------|------|
| 日常翻译 | gpt-3.5-turbo | 快速、准确 |
| 专业文档 | gpt-4 | 更准确的术语翻译 |
| 长文本 | claude-3-haiku | 上下文理解更好 |
| 文学作品 | claude-3-opus | 文采更好 |

### 多模型配置

您可以配置多个翻译服务，使用不同的模型：

1. 添加多个 OpenAI 翻译服务
2. 为每个服务配置不同的模型
3. 在翻译时可以切换使用

## 常用设置

### 快捷键自定义

在 Bob 设置 > 通用 中可以自定义快捷键：

- **划词翻译**：建议 `⌥ + D`
- **截图翻译**：建议 `⌥ + S`  
- **输入翻译**：建议 `⌥ + Space`
- **显示/隐藏**：建议 `⌥ + B`

### 翻译行为设置

优化翻译体验的推荐设置：

1. **自动识别语言**：开启
2. **翻译后自动复制**：根据需求开启
3. **保留原文格式**：开启（保持换行等格式）
4. **历史记录**：开启（方便查看翻译历史）

### 界面设置

- **主题**：选择适合的浅色或深色主题
- **字体大小**：根据屏幕调整舒适的大小
- **窗口位置**：设置弹出位置（跟随鼠标/固定位置）

## 使用技巧

### 1. 专业领域翻译

对于特定领域的翻译，可以在提示词中说明：

```text
请作为计算机专业译者，翻译以下技术文档。
保留所有技术术语的英文，用括号标注中文含义。
```

### 2. 批量翻译

需要翻译大量文本时：

1. 使用输入翻译模式
2. 将文本分段粘贴
3. 利用历史记录功能查看所有翻译

### 3. 对照阅读

阅读外文资料时：

1. 开启"显示原文"选项
2. 使用划词翻译实时查看
3. 对比原文和译文学习

### 4. 术语库管理

建立个人术语库：

1. 在常用术语翻译后添加到收藏
2. 自定义特定词汇的翻译
3. 导出术语库备份

## 常见问题

### 翻译速度慢

**可能原因**：
- 网络连接问题
- 选择的模型较大
- API 服务繁忙

**解决方案**：
1. 检查网络连接
2. 尝试使用 gpt-3.5-turbo 等快速模型
3. 避开高峰时段使用

### 翻译不准确

**改进方法**：
1. 使用更高级的模型（如 GPT-4）
2. 优化提示词，提供更多上下文
3. 对于专业内容，在提示词中说明领域

### API 配额用尽

**处理方式**：
1. 检查 API易 账户余额
2. 合理使用不同模型控制成本
3. 设置每日使用限制

## 最佳实践

### 1. 成本控制

- 日常翻译使用 gpt-3.5-turbo
- 重要文档才使用 gpt-4
- 定期查看使用统计

### 2. 翻译质量

- 提供充足的上下文
- 使用专业术语时说明领域  
- 对重要内容进行二次校对

### 3. 工作流优化

- 设置常用语言对
- 自定义专业领域提示词
- 善用历史记录和收藏功能

### 4. 数据安全

- 不翻译包含敏感信息的文本
- 定期清理翻译历史
- 妥善保管 API 密钥

## 进阶功能

### 集成工作流

Bob 支持通过 URL Scheme 集成到其他应用：

```bash
# 直接翻译文本
bob://translate?text=Hello&from=en&to=zh

# 打开 Bob 窗口
bob://open
```

### 自动化翻译

使用 AppleScript 实现自动化：

```applescript
tell application "Bob"
    translate "Hello World" from "en" to "zh"
end tell
```

### 导出功能

定期导出翻译记录：

1. 进入历史记录
2. 选择时间范围
3. 导出为 CSV 或 JSON 格式
