# Cherry Studio 集成指南

Cherry Studio 是一款功能强大的 AI 对话客户端，支持多种大语言模型。通过 API易，您可以在 Cherry Studio 中使用各种主流 AI 模型。

## 配置步骤

### 1. 获取 API 密钥

首先，您需要在 [API易控制台](https://vip.apiyi.com) 获取您的 API 密钥。

### 2. 打开设置

在 Cherry Studio 应用程序中：
1. 点击左下角的设置图标
2. 选择"模型提供商"选项

### 3. 配置 API易

1. 在模型提供商列表中选择 **AiHubMix** 或 **自定义**
2. 在 API 密钥字段中输入您的 API易 密钥
3. API 地址设置为：`https://api.apiyi.com`

### 4. 添加模型

1. 点击"添加模型"按钮
2. 从 API易 控制台复制您想使用的模型 ID
3. 将模型 ID 粘贴到 Cherry Studio 中
4. 设置模型别名（可选）

## 常用模型配置

以下是一些常用模型的配置示例：

| 模型名称 | 模型 ID | 说明 |
|---------|---------|------|
| GPT-4 | gpt-4 | OpenAI 最强大的模型 |
| Claude 3 | claude-3-opus-20240229 | Anthropic 的顶级模型 |
| GPT-3.5 | gpt-3.5-turbo | 快速且经济的选择 |

## 高级设置

### 图片支持

如果您需要使用支持图片输入的模型（如 GPT-4V），请：
1. 在设置中手动开启"图片"选项
2. 确保选择的模型支持视觉功能

### 流式输出

Cherry Studio 默认支持流式输出，无需额外配置。

## 故障排除

### 连接失败

如果遇到连接问题：
1. 检查 API 密钥是否正确
2. 确认网络连接正常
3. 验证 API 地址是否正确设置为 `https://api.apiyi.com`

### 模型不可用

如果模型显示不可用：
1. 确认您的账户余额充足
2. 检查该模型是否在您的订阅计划内
3. 尝试使用其他模型 ID

## 最佳实践

1. **定期更新模型列表**：API易 会不断添加新模型，定期检查更新
2. **合理选择模型**：根据任务需求选择合适的模型，平衡性能和成本
3. **监控使用量**：通过 API易 控制台监控您的 API 使用情况
