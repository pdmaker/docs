# 沉浸式翻译集成指南

沉浸式翻译是一款优秀的浏览器翻译扩展，支持网页双语对照阅读。通过集成 API易，您可以使用强大的 AI 模型获得更准确、更自然的翻译效果。

## 安装扩展

### 支持的浏览器

- Chrome / Edge / Brave
- Firefox
- Safari

### 安装步骤

1. 访问对应浏览器的扩展商店
2. 搜索"沉浸式翻译"或"Immersive Translate"
3. 点击安装并添加到浏览器

或访问 [沉浸式翻译官网](https://immersive-translate.owenyoung.com/) 获取安装链接。

## 配置 API易

### 步骤 1：打开设置

1. 点击浏览器工具栏的沉浸式翻译图标
2. 选择"设置"或点击设置图标
3. 在左侧菜单中选择"翻译服务"

### 步骤 2：选择自定义 API

1. 在翻译服务列表中找到"OpenAI"
2. 点击"管理"或"设置"按钮
3. 选择"自定义 API Key"选项

### 步骤 3：配置 API 参数

填写以下信息：

- **APIKEY**：输入您的 API易 密钥
- **自定义 API 接口地址**：`https://api.apiyi.com/v1/chat/completions`
- **自定义模型**：可选，如 `gpt-3.5-turbo` 或 `gpt-4`

点击"保存"完成配置。

## 使用方法

### 网页翻译

#### 自动翻译
1. 访问外文网页
2. 插件会自动检测并提示是否翻译
3. 点击翻译按钮开始翻译

#### 手动翻译
1. 点击浏览器工具栏的插件图标
2. 选择"翻译本页"
3. 等待翻译完成

### 翻译模式

#### 双语对照
默认模式，原文和译文并排显示：
- 保留原文格式
- 译文显示在原文下方
- 方便对照学习

#### 仅译文
只显示翻译后的内容：
- 完全替换原文
- 适合快速阅读
- 可随时切换回双语

### 划词翻译

1. 选中需要翻译的文本
2. 点击出现的翻译按钮
3. 在弹窗中查看翻译结果

## 高级设置

### 翻译引擎配置

优化不同类型内容的翻译效果：

```json
{
  "provider": "openai",
  "apiKey": "您的API易密钥",
  "apiUrl": "https://api.apiyi.com/v1/chat/completions",
  "model": "gpt-3.5-turbo",
  "systemPrompt": "你是一位专业的翻译专家，请提供准确、流畅的翻译。"
}
```

### 自定义提示词

针对不同内容类型的提示词模板：

#### 技术文档
```text
作为技术文档翻译专家，请：
1. 保留所有技术术语的原文
2. 在括号中提供中文解释
3. 保持代码和命令的原始格式
```

#### 学术论文
```text
作为学术翻译专家，请：
1. 使用学术规范的表达方式
2. 保留引用格式
3. 准确翻译专业术语
```

#### 文学作品
```text
作为文学翻译专家，请：
1. 保持原文的文学美感
2. 注意文化背景的转换
3. 保留修辞手法的效果
```

### 翻译规则

设置哪些网站自动翻译：

1. 进入设置 > 翻译规则
2. 添加网站域名
3. 选择翻译行为：
   - 总是翻译
   - 从不翻译
   - 智能判断

## 功能特色

### 1. PDF 翻译

支持在线 PDF 文档翻译：
- 保持 PDF 格式
- 支持双语对照
- 可复制译文

### 2. 视频字幕翻译

支持主流视频网站：
- YouTube
- Netflix
- Bilibili
- 等等

设置方法：
1. 开启"视频字幕翻译"
2. 选择字幕显示方式
3. 调整字幕样式

### 3. 电子书翻译

支持 EPUB 电子书：
1. 上传 EPUB 文件
2. 选择翻译设置
3. 下载双语版本

### 4. 输入框翻译

在网页输入框中实时翻译：
1. 在输入框输入文本
2. 按快捷键触发翻译
3. 查看翻译建议

## 常见问题

### 翻译失败

**可能原因**：
- API 密钥无效
- 网络连接问题
- 页面结构特殊

**解决方案**：
1. 验证 API 密钥是否正确
2. 检查 API 地址设置
3. 尝试刷新页面
4. 查看控制台错误信息

### 翻译速度慢

**优化方法**：
1. 使用更快的模型（如 gpt-3.5-turbo）
2. 减少单次翻译的文本量
3. 开启缓存功能
4. 检查网络连接

### 格式错乱

**处理方式**：
1. 尝试不同的翻译模式
2. 调整译文显示设置
3. 针对特定网站自定义规则
4. 报告问题给开发者
